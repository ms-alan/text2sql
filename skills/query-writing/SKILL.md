# Query Writing Skill

## Workflow for Simple Queries
1. Identify the table → Get schema → Write query → Execute

## Workflow for Complex Queries
1. Use `write_todos` to break down the task
2. Examine schemas for EACH table
3. Construct query with proper JOINs
4. Validate and execute
