# Schema Exploration Skill

## Workflow

### 1. List All Tables
Use `sql_db_list_tables` to see all available tables.

### 2. Get Schema for Specific Tables
Use `sql_db_schema` to examine columns, data types, sample data.

### 3. Map Relationships
Identify how tables connect through foreign keys.
